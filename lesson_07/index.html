While creating an application or working with files in your application, you will want to store a file in a folder. Sometimes there will be instances where the folder does not exist.

You want to create that folder during the process so the file can be stored without making the user resend the request.

Syntax

File::makeDirectory($path, $mode, $recursive, $force)

Parameters

The Fille::makeDirectory function takes the following as input parameters:

  • $patch This string represents the path of the directory we wish to create.

  • $modeThis is a four-digit integer that is used to set permissions for the directory.

  • $modeThis is a boolean variable used to specify if the recursive mode is set.

  • $forseThis is a boolean variable that, when set, lets you create a file in the profile path even when the directories in the path do not exist.

Return value

The function returns true if the requested directory is created. Otherwise, it returns false.
